# Codex 開発スタートガイド

このプロジェクトは、**計算ドリル** を新規構築するためのリポジトリです。

## 目的
- 小学生向けの短時間反復型計算ゲームを開発する
- まずは無料版MVPを完成させる
- Git を細かく刻みながら進める
- 各作業単位で commit と tag を残す

## Codex に最初にやってほしいこと
1. この `docs/codex/` 配下の文書をすべて読む
2. 現在ブランチと git status を確認する
3. 今回の作業対象フェーズを確認する
4. 変更前に、影響範囲のファイルを短く要約する
5. 実装する
6. `npm run build` で確認する
7. 変更内容の要約、想定 commit メッセージ、日本語タグ本文案を出す

## 現時点の前提
- 旧プロジェクトの継ぎ足しではなく、新規構築
- Phase 0 / 0 は完了済み想定
- 次の実作業は `Phase 0 / 1 of 5`

## Codex への依頼スタイル
- 1回で大きく進めすぎない
- **1 commit で収まる粒度** で進める
- ビルドエラーが出たら、その修正まで同じ作業単位で完結する
- 勝手に仕様を拡張しない
- 不明点は、まず既存 docs を読み、必要最小限の質問だけを行う

## 完了時に必ず出してほしいもの
- 変更ファイル一覧
- 変更概要
- 実施した確認内容
- 日本語 commit メッセージ
- 詳細付き tag 本文案
