# プロジェクト概要

## アプリ名
計算ドリル

## コンセプト
- 小学生向けの短時間反復型計算ゲーム
- 1プレイを短くし、何度もリトライしたくなる体験を重視
- 主軸は「短時間で何問正解できるか」
- 問題生成・正誤判定・記録集計は共通ロジックで管理する

## 初期版の主な仕様
- 演算: 足し算 / 引き算（掛け算・割り算はロック表示）
- モード: 時間制限モード
- 初期デフォルト: 10秒 / 初級 / 足し算 + 引き算
- 難易度: 初級のみ有効
- 正の整数のみ
- 引き算は負の答えを出さない

## 画面
- Top
- Settings（くわしい設定）
- Play
- Result
- Score（せいせき）

## 重要な体験方針
- 正解時はポップアップを出さない
- 正解時はタイマーを止めない
- 不正解時のみ短時間フィードバックを表示し、その間だけタイマー停止
- 現在問題が一目で分かる UI を最優先

## 後回しにするもの

- 無制限モード
