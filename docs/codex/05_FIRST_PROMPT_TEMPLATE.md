目的は **プロジェクト骨組みとダミーページの追加** です。

やってほしいこと:

6. 最後に以下を提示してください

   - 変更内容要約
   - 日本語 commit メッセージ案
   - tag 名案
   - 詳細付き tag 本文案

- 1 commit に収まる粒度で進めること
